Runtime attestation key.

The runtime attestation key (RAK) represents the identity of the enclave and can be used to sign remote attestations. Its purpose is to avoid round trips to IAS for each verification as the verifier can instead verify the RAK signature and the signature on the provided AVR which RAK to the enclave.
